## หมายเหตุ
- หากไม่มีรูปภาพ ระบบจะใช้ placeholder อัตโนมัติ
- รูปภาพควรมีขนาดประมาณ 400x300px สำหรับคอร์ส
- รูปภาพครูมวยควรเป็นรูปสี่เหลี่ยมจัตุรัส (1:1) สำหรับแสดงเป็นรูปวงกลม
